Usage¶

ProjectAssistant.py¶

  • ./ProjectAssistant.py -p | --publish
        Call the publisher to compile code documentation with the Sphinx documentation publisher.

  • ./ProjectAssistant.py -u | --push
        Begin the process of adding, committing, and pushing updates to GitHub via a series of git instructions.

Publisher.py¶

  • ./Administration/Documentum/Publisher/Author/Publisher.py
        Call the publisher to create documentation.
